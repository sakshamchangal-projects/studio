# **App Name**: VoiceGuard

## Core Features:

- Audio Analysis: Accepts Base64-encoded MP3 audio and analyzes its characteristics.
- Language Detection: Automatically detects the language of the audio sample (Tamil, English, Hindi, Malayalam, or Telugu).
- AI/Human Classification: Uses AI to classify the audio as AI_GENERATED or HUMAN.
- Confidence Scoring: Provides a confidence score (0.0 to 1.0) indicating the certainty of the classification.
- JSON Response: Returns the classification result (AI_GENERATED or HUMAN) and confidence score in a structured JSON format.
- API Key Authentication: Requires a valid API key for access.

## Style Guidelines:

- Primary color: Dark Blue (#224271) to inspire trust and security.
- Background color: Light grey (#f0f2f5) for a clean and professional interface.
- Accent color: Teal (#26a69a) to highlight key information.
- Body and headline font: 'Inter' (sans-serif) for readability and a modern feel.
- Use clear and concise icons to represent audio processing and results.
- A clean, straightforward layout to present the classification results clearly.
